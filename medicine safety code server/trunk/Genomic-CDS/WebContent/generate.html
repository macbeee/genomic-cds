Upload file in 23andMe format to generate a Medicine Safety Code

You can upload a file with genetic data in the 23andMe file format to generate a Medicine Safety Code. Examples of data in the 23andMe format can be found here.

Please select a file below. Files should not be compressed and their size should not exceed 50 MB. Uploading and processing the file may take some time.

Please note that most direct-to-consumer genetic testing services are not certified to provide data for guiding clinical decision making. Data from these testing services should not be used for such purposes.
